CURSOS OPINIÕES QUEM-SOMOS MAPA CONTATO

Um pouco sobre o projeto

Este site de curso foi desenvolvido em 2018 no curso de Sistemas para Internet no Instituto Federal do Triângulo Mineiro, buscando disseminar o conhecimento com aqueles interessantos na área de TI.


Fundamentos de Web Design

Promover o desenvolvimento da reflexão acerca dos conceitos inerentes à área do design, enfatizando os aspectos relacionados ao design para web. Capacitar o aluno na área de web design, com domínio da prática e de desenvolvimento de páginas web, utilizando aplicativos para criação de sites.


Arquitetura e Organização de Computadores

Este conteudo tem o objetivo em instruir iniciantes na área de TI, com o intuito de transmitir um básico ensino em Arquitetura e Organização de Computadores o qual demonstra o funcionamento a fundo de uma maquina.


Inglês Instrumental

Este é um curso indicado para pessoas iniciantes que desejam aprender sobre técnicas de leituras que podem ser úteis tanto no dia a dia quanto em provas e concursos públicos.


Metodologia Científica

Este conteúdo tem o intuito de apresentar um curso o qual tem o objetivo de mostrar uma forma de apresentação fromal de uma pesquisa, produto ou de uma criação autoral.


Sistemas Operacionais

Operar os sistemas padrão Windows e Linux, explorando seus recursos, provendo segurança, e realizando configurações que garantam o bom funcionamento da rede.


Lógica de Programação

Este conteúdo é direcionado aos iniciantes em programação, nas video-aulas utilizaremos a linguagem utilizada é JAVA pois é atualmente muito utilizada em Maratonas e no Mercado de Trabalho.
